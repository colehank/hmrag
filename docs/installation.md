# Installation

## Stable release

To install hmrag, run this command in your terminal:

```sh
uv add hmrag
```

Or if you prefer to use `pip`:

```sh
pip install hmrag
```

## From source

The source files for hmrag can be downloaded from the [Github repo](https://github.com/colehank/hmrag).

You can either clone the public repository:

```sh
git clone https://github.com/colehank/hmrag
```

Or download the [tarball](https://github.com/colehank/hmrag/tarball/main):

```sh
curl -OJL https://github.com/colehank/hmrag/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd hmrag
uv sync
```
